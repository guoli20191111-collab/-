# 02 · Step1b 选择角色

> 页面文件：`step1b.html`
> 页面路径：Step1 选择剧集 → **Step1b 选择角色** → Step2 上传照片
> 设计原则：瀑布流卡片展示，点击即选

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉（可切换剧集）+ 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 标题行 | 「选择你的古风形象吧」 + 角色数量 + 返回按钮 |
| 瀑布流角色卡片 | Grid 网格布局，9:16 竖版卡片，展示角色图片 |
| 卡片选中态 | 选中卡片金色边框 + ✦ 标记，其他卡片变暗 |
| 背景 | 模糊古风背景图 `assets/bg.PNG` |
| 底部进度条 | 进度 44% |

### 角色数据

| 剧集 | 角色列表 |
|------|---------|
| 《长相思》 | 小夭、相柳、涂山璟、玎嬙、轩辕辞、阿念、璇玑、涂山容、白子画、冯虞（共10位） |
| 《梦华录》 | 赵盼儿、顾千帆、宋引章、孙三娘、欧阳旭、柯旭（共6位） |
| 《琅琊榜》 | 梅长苏、靖王、霓凰、蒙挚、飞流、夏冬、誉王（共7位） |

### 交互逻辑

1. 进入页面 → 读取 `localStorage` 中已选剧集，渲染对应角色列表
2. 点击角色卡片 → 存入 `localStorage`，自动跳转 `step2.html`
3. 导航栏切换剧集 → 重新渲染瀑布流
4. 点击「返回」→ 回到 `step1.html`

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `selectedCharacter` | `0` ~ `N` | 选中角色索引 |
| `selectedCharacterName` | `"小夭"` | 选中角色名 |
