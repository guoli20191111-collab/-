# 07 · 选择图片版本

> 页面文件：`step4.html` 内弹窗 / 独立页面
> 页面路径：登录成功 → **选择图片版本** → Step5 生成同人文
> 设计原则：对比清晰，付费有感知

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 免费版 | 720p 带水印，可下载 |
| 付费版 | 1080p 高清无水印，¥6.6，可下载 |
| 版本对比 | 两个版本并排对比，差异一目了然 |
| 下载按钮 | 免费版直接下载，付费版需先支付 |
| 继续按钮 | 选择完成后进入下一步（生成同人文） |

### 交互逻辑

1. 登录成功后 → 弹出/进入此页面
2. 默认展示两个版本卡片并排对比
3. 点击「免费下载」→ 下载 720p 带水印图片
4. 点击「¥6.6 解锁高清」→ 拉起支付 → 支付成功 → 下载 1080p 无水印图片
5. 下载完成后「继续生成同人文」按钮可点击
6. 也可跳过下载，直接点击「继续」进入下一步

---

### 方案要点

- 点击高清版 → 水印消失，图片变清晰
