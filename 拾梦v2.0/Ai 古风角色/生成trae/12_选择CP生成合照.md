# 10 · Step6 选择 CP + 生成 CP 合照

> 页面文件：`step6_cp.html`（新建）
> 页面路径：同人文结果 → **Step6 选择 CP + 生成合照** → CP 合照预览
> 设计原则：CP 感拉满，选择即心动

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 步骤标题 | 「✦ 第四步 · 生成 CP 合照」 |
| 左侧我的形象 | 用户古风形象 9:16 展示 + 花名 |
| 中间连线动画 | 爱心连线动画，选中后显示 |
| 右侧 CP 预览 | 选中角色 9:16 大图展示 + 角色名 |
| 底部候选横滑 | 当前剧集所有角色小卡片横向滑动 |
| 「生成 CP 合照」按钮 | 选中后激活，点击调用 AI |

### 交互逻辑

1. 进入页面 → 左侧展示用户古风形象，右侧默认空或第一位角色
2. 点击底部候选卡片 → 右侧大图更新，爱心连线动画触发
3. 点击「生成 CP 合照」→ 调用 AI 生成
4. 等待生成 → 水墨动画等待态
5. 生成成功 → 跳转 CP 合照预览（11）

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `selectedCPName` | `"涂山璟"` | 选中的 CP 角色名 |
| `selectedCPIndex` | `2` | CP 角色索引 |
| `generatedCPImage` | 图片 URL | AI 生成的 CP 合照 |

### 方案要点
- 上方左右对称：我的形象（9:16）+ 爱心连线动画 + CP 形象（9:16）
- 图片下方**不显示名字**，鼠标悬停时才浮现角色名（hover tooltip 效果）
- 两侧模糊背景填充
- 底部候选列表横向滑动，选中高亮金边 ✦
- 未选中角色暗淡处理
- 「生成 CP 合照」按钮选中后激活
- 无 SVG 装饰
