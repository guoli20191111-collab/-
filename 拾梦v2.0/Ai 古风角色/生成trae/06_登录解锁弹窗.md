# 06 · 登录解锁弹窗

> 页面文件：`step4.html` 内弹窗组件
> 页面路径：Step4 满意结果 → **登录解锁弹窗** → 选择图片版本
> 设计原则：轻量弹窗，快速转化

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 弹窗遮罩 | 半透明黑色遮罩覆盖全屏，点击遮罩不可关闭（强制登录） |
| 引导文案 | 「保存你的古风形象，开启专属故事」 |
| 手机号登录 | 输入框 + 获取验证码 + 登录按钮 |
| 微信一键登录 | 微信图标按钮，一键授权 |
| 分隔线 | 「——— 或 ———」分隔两种登录方式 |
| 关闭按钮 | 右上角 × 可关闭弹窗（关闭后仍停留在结果页，但无法进入下一步） |

### 触发方式

| 触发来源 | 说明 |
|---------|------|
| 导航栏点击「未登录」 | 任意页面均可触发，未登录时导航栏右侧显示默认头像 + 「未登录」文字 |
| Step4 点击「满意，继续」 | 未登录时自动弹出，阻断继续流程 |

### 交互逻辑

1. 用户在 Step4 点击「满意，继续」/ 点击导航栏「未登录」→ 弹出此弹窗
2. 手机号登录流程：输入手机号 → 点击「获取验证码」→ 输入验证码 → 点击「登录」
3. 微信登录流程：点击「微信一键登录」→ 拉起微信授权 → 授权成功
4. 登录成功 → 弹窗关闭，自动进入「选择图片版本」（07）
5. 点击 × → 弹窗关闭，停留在结果页，再次点击「满意，继续」可重新弹出
