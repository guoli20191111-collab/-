# 08a · Step5a 输入花名

> 页面文件：`step5a.html`（新建）
> 页面路径：选择图片版本（07）→ **输入花名** → 选择关系设定（08b）
> 设计原则：轻巧入戏，花名即角色代入感的开始

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 左侧古风形象 | 9:16 居中展示，两侧模糊填充 |
| 步骤标题 | 「✦ 第三步 · 生成同人文」 |
| 花名输入框 | 用户自定义花名，支持手动输入 |
| 随机骰子按钮 | 点击 🎲 随机生成一个古风花名 |
| 「下一步」按钮 | 花名非空时激活，跳转关系设定（08b） |

### 交互逻辑

1. 进入页面 → 左侧展示古风形象，右侧显示花名输入框
2. 用户输入花名 / 点击 🎲 随机生成
3. 花名非空 → 「下一步」按钮激活
4. 点击「下一步」→ 存储花名 → 跳转 step5b.html

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `characterFlowerName` | `"云笙"` | 用户输入或随机生成的花名 |
