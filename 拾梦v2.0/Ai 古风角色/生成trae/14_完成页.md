# 12 · 完成页

> 页面文件：`complete.html`（新建）
> 页面路径：CP 合照预览 → **完成页**
> 设计原则：温暖收尾，情感达到顶峰，引导再来

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 金色粒子动画 | 进入页面自动播放，1~2 秒后渐隐 |
| 完成标题 | 「✦ 你的古风之旅已完成 ✦」+ 副标题 |
| 成果缩略图 | 三张成果横向排列：古风形象 / 同人文截图 / CP 合照 |
| 「再来一次」按钮 | 清除数据，回到首页重新体验 |
| 「分享全旅程」按钮 | 生成旅程长图，跳转小红书发布 |

### 交互逻辑

1. 进入页面 → 金色粒子飘落动画自动播放
2. 动画渐隐 → 展示完成标题 + 三张成果缩略图
3. 点击「再来一次」→ 清除 localStorage → 回到 `index.html`
4. 点击「分享全旅程」→ 生成旅程长图 → 跳转小红书发布

### 数据读取

| Key | 来源 | 说明 |
|-----|------|------|
| `generatedImage` | Step3 存入 | 古风形象缩略图 |
| `generatedNovel` | Step5 存入 | 同人文（截图用） |
| `generatedCPImage` | Step6 存入 | CP 合照缩略图 |
| `characterFlowerName` | Step5 存入 | 花名 |
| `selectedCPName` | Step6 存入 | CP 角色名 |
