# 09 · 同人文结果 + 解锁方式

> 页面文件：`step5_result.html`（新建）
> 页面路径：Step5 生成同人文 → **同人文结果 + 解锁** → Step6 生成 CP 合照
> 设计原则：先尝后买，解锁即继续

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 缩略图 + 标题 | 左侧小缩略图，右侧标题 + 副标题，右上角三个图标按钮 |
| 图标按钮组 | 复制 / 分享 / 下载，纯图标无文字 |
| 正文展示 | 可滚动，300 字后文字渐隐模糊 |
| 底部两个主按钮 | `✦ ¥9.9 解锁` / `继续生成 CP 合照 →` |

### 交互逻辑

1. 进入页面 → 展示同人文前 300 字，末尾渐隐模糊
2. 点击「¥9.9 解锁」→ 拉起支付 → 成功后全文展开，模糊消失
3. 点击「继续生成 CP 合照」→ 跳转 Step6（10）
4. 解锁后图标按钮可用：复制全文 / 分享 / 下载

### 数据读取

| Key | 来源 | 说明 |
|-----|------|------|
| `generatedNovel` | Step5 存入 | AI 生成的同人文全文 |
| `generatedNovelTitle` | Step5 存入 | 同人文标题 |
| `selectedCharacterName` | Step1b 存入 | 角色名 |
| `characterFlowerName` | Step5 存入 | 花名 |
