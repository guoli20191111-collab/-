# 01 · Step1 选择剧集

> 页面文件：`step1.html`
> 页面路径：首页 → **Step1 选择剧集** → Step1b 选择角色
> 设计原则：全屏视频沉浸，顶部导航操作

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | 左侧 Logo（点击回首页）、中部「IP 剧集」下拉框、右侧头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 背景视频轮播 | 3 段古风视频自动循环播放（与首页共用素材） |
| IP 剧集下拉框 | 点击展开，列出 3 部可选古装剧，选中后高亮并跳转选角色页 |
| 头像下拉菜单 | 点击头像弹出：显示「未登录」+「去登录」按钮 |
| 底部进度条 | 金色渐变进度条，当前进度 33% |

### 交互逻辑

1. 默认高亮第一部剧《长相思》
2. 点击「IP 剧集 ▾」→ 展开下拉列表，显示 3 部剧名
3. 点击某部剧名 → 存入 `localStorage`，自动跳转 `step1b.html`
4. 点击 Logo → 返回首页
5. 点击头像 → 弹出登录下拉

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `selectedDrama` | `{"label":"《长相思》"}` | 选中的剧集对象 |
| `selectedDramaIndex` | `0` / `1` / `2` | 剧集索引 |
