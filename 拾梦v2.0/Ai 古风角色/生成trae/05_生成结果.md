# 05 · Step4 生成结果

> 页面文件：`step4.html`
> 页面路径：Step3 AI 生成等待 → **Step4 生成结果** → 登录解锁弹窗
> 设计原则：结果冲击，满意即留

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 生成结果展示 | 全屏展示 AI 生成的古风形象大图 |
| 满意/不满意操作 | 用户确认是否满意结果 |
| 重新生成 | 不满意时可返回重新生成 |
| 全屏查看 | 点击图片可全屏查看 |
| 背景 | 纯黑底，突出生成图片 |

### 交互逻辑

1. 进入页面 → 读取 `localStorage` 中 `generatedImage` 展示
2. 用户满意 → 弹出登录解锁弹窗（06）
3. 用户不满意 → 返回 `step2.html` 重新上传/生成
4. 点击图片 → 全屏查看，点击关闭退出

### 数据读取

| Key | 来源 | 说明 |
|-----|------|------|
| `generatedImage` | Step3 存入 | AI 生成的图片 |
| `userPhoto` | Step2 存入 | 头像展示用 |

### 方案要点
- 左侧 60%：9:16 图片居中，两侧同图模糊填充，不裁切主体

- 「满意，继续」金色主按钮独占一行
- 「重新生成」和「下载保存」并排为次按钮
