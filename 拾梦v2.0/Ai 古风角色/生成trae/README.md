# 拾梦录 · 页面文档总览

> 古风 AI 角色生成 Web App
> 设计原则：暗金古风，沉浸体验，先尝后买

---

## 用户完整流程

```
首页
 └→ 选择剧集
     └→ 选择角色
         └→ 上传照片
             └→ AI 生成等待
                 └→ 生成结果
                     └→ 登录解锁弹窗
                         └→ 选择图片版本
                             └→ 输入花名
                                 └→ 选择关系设定
                                     └→ 选择剧情走向
                                         └→ 同人文结果与解锁
                                             └→ 选择 CP 生成合照
                                                 └→ CP 合照预览与操作
                                                     └→ 完成页
```

## 设计规范

| 项目 | 规范 |
|------|------|
| 背景色 | `#08060f` |
| 金色主色 | `rgba(200,160,100,x)` |
| 字体 | Noto Serif SC |
| 图片比例 | 9:16（居中 + 两侧同图模糊填充） |
| 图标库 | Lucide / Heroicons |
| 导航栏 | Logo + 剧集下拉 + 头像，毛玻璃效果 |
| 导航栏·未登录 | 默认头像图标 + 「未登录」文字，点击 → 弹出登录弹窗（06） |
| 导航栏·已登录 | 用户照片头像，点击可查看账户信息 |
| 装饰 | 无 SVG 角落装饰 |
