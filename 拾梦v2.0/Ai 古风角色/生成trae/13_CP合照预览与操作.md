# 11 · CP 合照预览与操作

> 页面文件：`step7_cp_result.html`（新建）
> 页面路径：Step6 生成 CP 合照 → **CP 合照预览** → 完成页（12）
> 设计原则：延续 Step4 布局风格，左图右面板，CP 感与操作感并存

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 左侧 CP 合照 | 9:16 居中展示，两侧模糊填充，点击可全屏查看 |
| 右侧信息面板 | 毛玻璃面板，CP 名 + 剧集 + 操作按钮 |
| 「完成，去收藏」按钮 | 主按钮，跳转完成页（12） |
| 分享 / 下载按钮 | 次级操作，并排展示 |

### 交互逻辑

1. 进入页面 → 读取 `generatedCPImage` 展示 CP 合照
2. 点击合照 → 全屏查看，点击空白或 × 退出
3. 点击「分享」→ 调用系统分享 / 小红书
4. 点击「下载」→ 下载 CP 合照到本地
5. 点击「完成，去收藏」→ 跳转完成页（12）

### 数据读取

| Key | 来源 | 说明 |
|-----|------|------|
| `generatedCPImage` | Step6 存入 | AI 生成的 CP 合照 |
| `selectedCPName` | Step6 存入 | 选中的 CP 角色名 |
| `characterFlowerName` | Step5 存入 | 用户花名 |
| `selectedDrama` | Step1 存入 | 剧集名称 |

### 方案要点
- 左侧 60%：CP 合照 9:16 居中，两侧同图模糊填充，点击可全屏
- 右侧 40%：毛玻璃面板，展示「花名 × CP名」标题 + 剧集名
- 「完成，去收藏」金色主按钮独占一行
- 「分享」和「下载」并排为次按钮（Lucide: Share2 / Download）
- 无 SVG 装饰
