# 04 · Step3 AI 生成等待

> 页面文件：`step3.html`
> 页面路径：Step2 上传照片 → **Step3 AI 生成等待** → Step4 生成结果
> 设计原则：水墨禅意，等待不焦虑

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 水墨扩散动画 | 三层同心圆脉冲扩散 + 中心旋转光晕，模拟墨水晕染 |
| 等待文案 | 「✦ 正在为你穿越到古代 ✦」，呼吸灯闪烁 |
| 时间提示 | 「预计需要 10 ~ 20 秒」 |
| 背景 | 模糊古风背景图 |

### 交互逻辑

1. 进入页面 → 自动调用 `config.js` 中的 `generateAncientStyleImage()` 接口
2. 等待过程 → 水墨动画持续播放
3. 生成成功 → 存入 `localStorage`，0.5s 后跳转 `step4.html`
4. 生成失败 → 弹出错误提示，1.5s 后返回 `step2.html`

### 数据读取

| Key | 来源 | 说明 |
|-----|------|------|
| `userPhoto` | Step2 存入 | 用户照片 base64 |
| `selectedDramaIndex` | Step1 存入 | 剧集索引 |
| `characterName` | Step1b 存入 | 角色名 |

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `generatedImage` | 图片 URL / base64 | AI 生成的古风形象图 |
| `generatedPrompt` | 文本 | 生成使用的 prompt |
| `generatedStyle` | 文本 | 生成的风格描述 |
