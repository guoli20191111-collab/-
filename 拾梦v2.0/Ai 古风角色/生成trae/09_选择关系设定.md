# 08b · Step5b 选择关系设定

> 页面文件：`step5b.html`（新建）
> 页面路径：输入花名（08a）→ **选择关系设定** → 选择剧情走向（08c）
> 设计原则：四选一，点击即心动

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 左侧古风形象 | 9:16 居中展示，两侧模糊填充 |
| 步骤标题 | 「✦ 选择关系设定」 |
| 四个关系选项 | 竹马 / 师徒 / 宿敌转恋人 / 自定义输入 |
| 「下一步」按钮 | 选中后激活，跳转剧情走向（08c） |

### 交互逻辑

1. 进入页面 → 显示花名回显「「云笙」与他的关系是…」
2. 点击选项卡片 → 高亮选中态（金边 ✦）
3. 选择「自定义」→ 展开文本输入框
4. 点击「下一步」→ 存储关系设定 → 跳转 step5c.html

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `novelRelationship` | `"竹马"` | 选中的关系设定 |
| `novelRelationshipCustom` | 文本 | 自定义关系文案 |

### 方案要点
- 左侧 60%：古风形象持续展示（与 08a 一致）
- 右侧 40%：毛玻璃面板，2×2 选项卡片布局
- 选中卡片金边高亮 ✦，其余暗淡
- 选「自定义」时卡片下方展开输入框
- 未选中时「下一步」置灰
- 无 SVG 装饰
