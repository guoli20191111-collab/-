# 00 · 首页（落地页）

> 页面文件：`index.html`
> 页面路径：首页 → Step1 选择剧集
> 设计原则：品牌沉浸，一键进入

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 背景视频轮播 | 3 段古风视频（`assets/shipin/1~3.mp4`）自动循环播放，叠加渐变暗色遮罩 |
| 品牌 Logo | 居中展示 `assets/logo.png`，带金色投影 |
| Slogan | 「旧梦可拾，此刻成录」，淡金色描边字 |
| 「开始体验」按钮 | 居中主按钮，点击跳转 `step1.html` |
| 四角 ✦ | 四角浮动 ✦ 符号，纯文字氛围点缀 |

### 交互逻辑

1. 进入页面 → 第一段视频自动播放
2. 视频播完 → 淡入淡出切换下一段，循环播放
3. 点击「开始体验」→ 跳转 `step1.html`
