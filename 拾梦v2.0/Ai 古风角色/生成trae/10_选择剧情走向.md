# 08c · Step5c 选择剧情走向

> 页面文件：`step5c.html`（新建）
> 页面路径：选择关系设定（08b）→ **选择剧情走向** → 生成等待 → 同人文结果（09）
> 设计原则：最后一步，选完即生成，期待感拉满

---

## 页面功能说明

| 功能项 | 说明 |
|--------|------|
| 顶部导航栏 | Logo + IP 剧集下拉 + 头像（未登录：默认头像 + 「未登录」文字，点击弹出登录弹窗；已登录：用户照片） |
| 左侧古风形象 | 9:16 居中展示，两侧模糊填充 |
| 步骤标题 | 「✦ 选择剧情走向」 |
| 四个走向选项 | 甜宠 / 虐恋 / 双向暗恋 / 自定义输入 |
| 「开始生成同人文」按钮 | 选中后激活，点击调用 AI 生成 |
| 生成等待态 | 按钮点击后切换为等待动画 |

### 交互逻辑

1. 进入页面 → 回显花名 + 关系设定，提示「这段故事的基调是…」
2. 点击选项卡片 → 高亮选中态（金边 ✦）
3. 选择「自定义」→ 展开文本输入框
4. 点击「开始生成同人文」→ 存储走向 → 调用 AI → 进入等待态
5. 生成成功 → 跳转同人文结果（09）
6. 生成失败 → 提示错误，可重试

### 数据存储

| Key | 值 | 说明 |
|-----|---|------|
| `novelPlotDirection` | `"甜宠"` | 选中的剧情走向 |
| `novelPlotCustom` | 文本 | 自定义走向文案 |
| `generatedNovel` | 文本 | AI 生成的同人文全文 |
| `generatedNovelTitle` | 文本 | 同人文标题 |

### 方案要点
- 左侧 60%：古风形象持续展示（与 08a/08b 一致）
- 右侧 40%：毛玻璃面板，2×2 选项卡片布局
- 选中卡片金边高亮 ✦，其余暗淡
- 选「自定义」时卡片下方展开输入框
- 未选中时「开始生成同人文」置灰
- 无 SVG 装饰
