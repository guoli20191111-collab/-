# 古风同人图·页面功能使用说明

> 产品：古风同人图 · 网页端（电脑浏览器）· MVP 阶段
> 面向：C 端用户
> 完整路径：选剧 → 上传照片+取花名 → AI生成等待 → 结果展示 → CP合照 / 同人文

## Step 1 · 选择古装剧

### 页面功能说明

本页是用户进入产品后的第一个交互页面，核心目的是让用户选定生成风格所对应的古装剧集世界观。

| 功能项 | 说明 |
|--------|------|
| 剧集卡片展示 | 横排展示 3 张热播古装剧海报卡片，每张包含剧名 + 一句话剧情简介 |

| 「下一步」按钮 | 选中剧集后点击「下一步」进入 Step2，未选中时按钮置灰不可点 |
| 选剧依据 | MVP 优先选当前热播或近期热播、社交媒体热度高、服饰风格辨识度强的剧集 |

## Step 2 · 上传照片 + 取花名

### 页面功能说明

用户上传本人照片，并为自己选取一个古风花名，花名将作为后续同人文中的主角名字。

**上传照片功能**

| 功能项 | 说明 |
|--------|------|
| 上传入口 | 点击上传区域或拖拽照片上传 |
| 支持格式 | JPG / PNG |
| 页面提示语 | 「上传一张能看清脸的照片，效果最佳」 |
| 拍摄建议 | 正面、光线充足 |
| 预览展示 | 上传成功后在区域内预览缩略图 |

**取花名功能**

| 功能项 | 说明 |
|--------|------|
| 自定义输入 | 用户可在输入框中自由填写古风名字（如「云笙」「若烟」） |
| 随机推荐 | 点击「随机推荐」，系统弹出 3 个古风名字供选择，点击即填入 |
| 花名用途 | 花名将出现在后续同人文中作为主角名 |

## Step 3 · AI 生成等待页

### 页面功能说明

用户提交照片和花名后，系统进行 AI 生成处理，此页面为生成过程中的等待反馈页。目的是降低用户等待焦虑感，提升沉浸感。

| 功能项 | 说明 |
|--------|------|
| 加载动画 | 古风水墨风格加载动画（如毛笔绘制圆圈、水墨晕染效果） |
| 等待提示文字 | 「正在为你穿越到古代…」 |
| 预计等待时长 | 10 ~ 20 秒 |
| 进度反馈 | 可配合进度条或逐步显示的提示文字增强等待感知 |
| 不可返回 | 生成过程中禁止返回操作，避免中断流程 |

**生成逻辑（后端 Dify 工作流）**

```
用户照片 + 选择的剧集
       ↓
节点1：Claude 分析服装颜色和风格
       ↓
节点2：根据分析结果组装提示词
       ↓
节点3：调用即梦 API 生成古风图
       ↓
输出：古风图片 URL → 跳转 Step4
```

## Step 5A · 生成 CP 合照

### 页面功能说明

用户选择想与之合拍的剧中角色，系统生成双人合照图。

**角色选择**

| 功能项 | 说明 |
|--------|------|
| 角色列表 | 展示当前所选剧集的角色列表（MVP 阶段限定女性角色，每部剧 3~4 个） |
| 角色卡片内容 | 角色图（漫改版）+ 角色名 + 一句话性格简介 |
| 示例 | 「女二·沈月」— 「温婉聪慧，深藏不露」 |
| 选中交互 | 点击高亮，可更换选择，确认后点击「生成合照」 |

**合照生成逻辑**

| 项目 | 说明 |
|------|------|
| 构图 | 用户古风图（右侧）+ 所选角色模板图（左侧），两人并肩站立 |
| 背景 | 与所选剧集场景风格一致 |
| 角色模板来源 | 运营预先用即梦生成的漫改版角色图（写实 CG 风格） |

**合照结果页操作**

| 按钮 | 功能 |
|------|------|
| 下载保存 | 下载合照到本地 |
| 分享小红书 | 一键跳转/生成小红书分享内容 |

## Step 5B · 生成同人文

### 页面功能说明

用户选择剧情方向，AI 基于所选剧集世界观和用户花名自动生成约 300 字的古风同人文。

**剧情方向选择**

| 方向 | 说明 |
|------|------|
| 爱恋甜蜜 | 生成温馨浪漫、情投意合的甜宠风格同人文 |
| 虐恋错过 | 生成命运弄人、错过遗憾的虐心风格同人文 |

**同人文规格**

| 项目 | 说明 |
|------|------|
| 字数 | 约 300 字 |
| 主角名 | 用户在 Step2 取的花名 |
| 世界观 | 基于用户所选剧集的背景设定 |
| 生成方式 | AI 一键生成，无需用户额外输入 |
| 技术实现 | Dify 工作流调用 Claude API 生成 |

**结果页操作**

| 按钮 | 功能 |
|------|------|
| 复制全文 | 一键复制同人文文字内容 |
| 分享小红书 | 一键跳转/生成小红书分享内容（图文合并） |
| 重新生成 | 保持相同方向重新生成一篇 |
| 换个方向 | 返回方向选择重新选择 |
